# 简易的信息管理系统
## 初始化：
#### cd demo根目录
#### npm run mocData
#### windows: move userList.js ./data/
#### like-unix: mv userList.js ./data/
## 运行：
#### node server.js
#### 默认3333端口，可修改server.js文件下的port变量以改变端口值，建议设置在在3000~65530之间， 如果有不明情况的404可以考虑换端口（原因可能是被占用或其他。）
## 访问:
#### http://host:port/userList.html
# Enjoy it!
